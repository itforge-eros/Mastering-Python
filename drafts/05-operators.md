# Operators
Operators หรือตัวดำเนินการ

| การดำเนินการ | วิธีการเขียน | หน้าที่ |
|:---:|:---:|:---:|
| Addition | + | บวก |
| Subtraction | - | ลบ |
| Multiplication | * | คูณ |
| Division | / | หาร |
| Floor Division | // | หารแบบตัดเศษ |
| Modulo | % | หารเพื่อเอาเศษ |
| Exponential | ** | ยกกำลัง |
